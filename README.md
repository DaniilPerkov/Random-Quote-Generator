# Random Quote Generator

**Автор:** [Даниил [Перков]

## Описание

Это простое GUI-приложение, написанное на языке Python с использованием библиотеки Tkinter. Приложение позволяет генерировать случайные цитаты из предопределенного списка, просматривать историю сгенерированных цитат, фильтровать их по автору или теме, а также добавлять собственные цитаты.

Данные о цитатах и истории сохраняются в формате JSON для обеспечения персистентности между запусками программы.

## Технологии

*   **Язык:** Python 3.x
*   **Библиотеки:** Tkinter (стандартная), json (стандартная), random (стандартная)
*   **Система контроля версий:** Git

## Инструкция по запуску

1.  Убедитесь, что у вас установлен Python 3.
2.  Склонируйте этот репозиторий или скачайте файлы.
3.  Откройте терминал в папке проекта.
4.  Запустите приложение командой:
    ```bash
    python quotes_app.py
    ```

## Использование приложения

1.  **Главное окно:** При запуске отображается пустое поле для цитаты и кнопка «Сгенерировать цитату».
2.  **Генерация:** Нажмите кнопку, чтобы получить случайную цитату. Она появится вверху окна и добавится в историю.
3.  **История:** В нижней части окна отображается список всех ранее сгенерированных цитат.
4.  **Фильтрация:** Используйте выпадающие списки над историей, чтобы отфильтровать цитаты по "Автору" или "Теме".
5.  **Добавление цитаты:**
    *   Введите текст цитаты, имя автора и тему в соответствующие поля в блоке "Добавить новую цитату".
    *   Нажмите кнопку "Добавить цитату".
    *   *Проверка:* Если поле "Текст", "Автор" или "Тема" пустое, программа покажет ошибку.
6.  **Сохранение:** Все добавленные пользователем цитаты сохраняются в файл `quotes.json` и будут доступны при следующем запуске.

## Примеры использования / Тесты

*   **Тест генерации:** Нажать кнопку "Сгенерировать цитату" > Проверить, что текст и автор появились на экране.
*   **Тест истории:** Сгенерировать 3 разные цитаты > Проверить, что список истории внизу содержит все три цитаты.
*   **Тест фильтрации:** Сгенерировать цитаты от разных авторов > Выбрать одного автора в фильтре > Проверить, что в списке остались только его цитаты.
*   **Тест добавления (Успех):** Ввести данные во все поля и нажать "Добавить" > Проверить появление новой кнопки в фильтрах и возможность сгенерировать новую цитату.
*   **Тест добавления (Ошибка):** Оставить поле "Тема" пустым и нажать "Добавить" > Должно появиться окно с ошибкой "Все поля должны быть заполнены".
